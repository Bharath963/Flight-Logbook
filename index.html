CLOUD DATABASE SETTINGS

Set up Free Cloud Sync (Firebase Firestore)

By default, all your flight logs are saved locally on your phone/browser. Configure a free Firebase Project below to back up and sync your flight ledger across all your devices.

How to get credentials:

  1. Go to the Firebase Console.
  2. Create a new project (under the Free Spark Plan).
  3. Register a new "Web App" inside your Firebase project.
  4. Copy the firebaseConfig script snippet and paste the key values below.
  5. In the Firebase Console sidebar, go to **Firestore Database** and click **Create Database**, starting in **Test Mode** (or configure public read/write rules).
